## Approach Taken

- 3 Roles. Admin to approved the Partners. Partners to provide services and User to hire our Partners.

**Partners**

- Create Account
- Log In
- Able to Edit profile, delete account
- Upload photo with cloudinary
- Hire History
- Display Testimonial

**User**
- Create Account
- Log In
- Able to Edit profile, delete account
- Hire Partner
- Give Testimonial
